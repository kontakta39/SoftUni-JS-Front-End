# 🎓 JavaScript Front-End – SoftUni Course Portfolio (February 2024)

## 📌 Overview

This repository contains my solutions and exercises completed during the [*JS Front-End - February 2024 @ SoftUni*](https://softuni.bg/trainings/4362/js-front-end-february-2024) course.

The course focuses on building a solid foundation in **JavaScript programming and front-end development**, covering core language concepts, DOM manipulation, and event-driven programming.

---

## 🚀 Key Learning Outcomes

Throughout the course, I gained practical experience in:

* Writing JavaScript using modern syntax and best practices
* Working with variables, data types, loops, and conditional logic
* Processing arrays, strings, and using regular expressions
* Implementing reusable logic through functions and arrow functions
* Working with objects, JSON, and ES6 classes
* Interacting with the browser using the DOM API
* Handling events and building interactive web behavior
